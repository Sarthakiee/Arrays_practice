## 🧠 Example Problem

### 🔹 Maximum Subarray Sum (Kadane’s Algorithm)

**Problem:**  
Find the contiguous subarray with the maximum sum.

**Approach:**  
Use Kadane’s Algorithm to maintain current sum and update maximum sum.

**Time Complexity:** `O(n)`  
**Space Complexity:** `O(1)`
